# Alura Latam

Esta web fue de mis primeras experiencias externas a mi carrera y me permitió formarme no solamente en el mundo de la programación si no en el desarrollo personal mediante técnicas de autodesarrollo, el ser productivo y el poder enfocarse día a día, también obtuve un certificado con el uso de LinkedIn y Git con GitHub.

## Lógica de Programación
- Muestra mensajes en pantalla de forma interactiva
- Utiliza variables en el desarrollo de software
- Adéntrate en el mundo del desarrollo siguiendo buenas prácticas de programación
- Desarrolla una aplicación de principio a fin, inspirada en el mundo real
- Aprende a adaptar soluciones proporcionadas por el lenguaje en tus programas
- Aprenda qué son las funciones y cómo utilizar JavaScript para manipular elementos en la pantalla
- Desarrolla aún más tu lógica de programación inspirándote en un proyecto real
- Aprende a usar Github para publicar tus proyectos
- Descubre cómo utilizar las listas, una de las principales estructuras de datos en el mundo de la programación
- Cree manipulaciones de palabras y números dinámicamente con JavaScript

## HTML y CSS
- Aprenda lo que es HTML y CSS
- Entienda la estructura básica de un archivo HTML
- Utilice el navegador para inspeccionar elementos
- Aprenda a definir estilos para elementos usando CSS
- Desarrolle una página Web desde cero
- Aprenda la estructura de la página HTML
- Navegue entre páginas Web
- Conozca el reset.css y el prosicionamiento por el CSS
- Entienda las diferencias entre inline-block
- Aprenda a lidiar con bordes y capturar eventos con CSS
- Cree encabezado, cuerpo y pié de página de una página Web
- Cree formularios complejos
- Trabaje con campos para dispositivos móviles
- Presente informacioes en tablas
- Use estilos para formularios, campos y tablas
- Entienda la jerarquía en el CSS
- Trabaje con transformaciones y transiciones
- Importe contenido externo a tu página HTML, como fuentes, videos y mapas
- Conozca las pseudoclases y los pseudoelementos
- Utilice selectores CSS avanzados
- Aprenda a lidiar con la opacidad y la sombra
- Comprenda Viewport y design responsivo


## Java
- Cree un proyecto Java usando IntelliJ
- Aprenda sobre el proceso de compilación de una aplicación Java
- Conozca los principales tipos de datos
- Controle el flujo de una aplicación con el uso de condicionales y bucles
- Aprende a representar una lista de objetos en Java
- Comprender cómo funciona la clase Java ArrayList
- Manipular una lista de objetos
- Ordenar una lista de acuerdo con algunos criterios
- Reparto de objetos y verificación de tipo con instanceof
- Aprende los conceptos de Orientación a Objetos usando Java
- Modelar abstracciones de aplicaciones a través de clases, atributos y métodos
- Usar la herencia como mecanismo de reutilización de código
- Aplicar polimorfismo para hacer que el código del proyecto sea flexible
- Comprender cómo usar interfaces para estandarizar métodos
- Aprenda a consumir una API HTTP en Java
- Comprender cómo recibir y convertir información de una API a objetos Java
- Manipular datos en formato JSON

- Utilice clases del paquete java.io para la manipulación de archivos
